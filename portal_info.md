## 🤫 隠し機能：シークレット・チャットルームの起動方法

社内のメンバーや管理者だけがアクセスできる、機密性の高い隠しチャットルームが実装されています。

### 1. アクセス方法
* ログイン後の画面の左側メニュー（サイドバー）の一番下にある **「ログアウト」ボタンのすぐ上** をマウスでホバー（指を乗せる）します。
* すると、通常時は背景と同化してほぼ見えない **「3つの点が線で結ばれた薄いマーク（星座のようなアイコン）」** が、明るく青色にふわっと浮かび上がります。
* そこをクリック（スマホならタップ）すると、**パスワードの入力プロンプト**が表示されます。
* 正しいパスワード **`secret123`** を入力すると、隠しタブである **「シークレットチャットルーム」** が起動します。

### 2. 特殊仕様（10分で自動消滅）
* このチャットに入力した発言は、**送信された瞬間から「ちょうど10分」が経過すると、画面から消えるだけでなく、ブラウザのデータベースからも物理的に完全消去（自動クレンジング）されます。**
* 10分以上前の古い会話は一切の痕跡を残さずに消え去るため、一時的な雑談や機密性の高いアイデア出しに最適です。

## 🚀 アプリケーションの起動とスマホ確認手順

### 1. パソコンのローカル環境での起動
PC内でWebサーバーを起動して確認する場合の手順です。（現在ポート `8000` でバックグラウンド起動中ですが、PC再起動時などは以下で再開できます）
* フォルダ `HP` を開き、PowerShell等のコマンドラインで以下を実行します。
  ```bash
  python -m http.server 8000
  ```
* ブラウザで **[http://localhost:8000](http://localhost:8000)** にアクセスします。

### 2. スマホから確認する（Netlify Dropを使用する手順）
ファイアウォールなどの影響を受けず、今すぐご自身のスマホで本物として動作させるための手順です。

1. パソコンのブラウザで **[Netlify Drop](https://app.netlify.com/drop)** にアクセスします（完全無料・登録不要）。
2. この `HP` フォルダ全体を、Netlifyの画面へそのままドラッグ＆ドロップします。
3. 3〜5秒で専用のURL（`https://xxx.netlify.app`）が発行されます。
4. そのURLをスマホに入力すれば、手元でログインや打刻アニメーション、チャットが完璧に動作します。

## 🔗 将来のGoogleスプレッドシート（本物のDB）連携手順
モックDBから、本物のスプレッドシートへ切り替える際は以下の手順を行います。

1. Googleスプレッドシートを新規作成し、下部に `Users`, `Attendance`, `Board`, `Schedule` の4つのシートを用意します。
2. アプリの **「システム設定」** タブに表示されている「GAS側のコード構成例」をコピーし、スプレッドシートの「拡張機能 ＞ Apps Script」に貼り付けてWebアプリとしてデプロイします。
3. **【管理者へのメール転送設定】**:
   スプレッドシートのGASの `doPost(e)` 内に、以下のコードブロックを追加することで、アドレスを露出させずに `snc23@outlook.jp` へ直接メールが送信されるようになります。
   ```javascript
   if (action === "sendMail") {
     var subject = e.parameter.subject;
     var body = e.parameter.body;
     var senderEmail = e.parameter.senderEmail;
     var senderName = e.parameter.senderName;
     
     MailApp.sendEmail({
       to: "snc23@outlook.jp",
       subject: "【ポータル問合せ】" + subject,
       body: "送信者: " + senderName + " (" + senderEmail + ")\n\n" + body
     });
     return ContentService.createTextOutput(JSON.stringify({success: true})).setMimeType(ContentService.MimeType.JSON);
   }
   ```
4. 発行されたGASのURLを、ポータル内の **「システム設定」＞「GAS Web App API URL」** に貼り付け、「有効化」にチェックを入れて保存します。
5. 「接続テスト」を押し、「接続成功」と出れば連携完了です。以降、ログインや打刻、および管理者への直接メール送信がすべてスプレッドシートおよびGAS経由で実機稼働します。
